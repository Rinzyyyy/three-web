
A 3D portfolio site built with React and Three.js. 
It presents skills and projects in an interactive 3D gallery on desktop and a scroll-based layout on mobile.


## Purpose

- **Present skills and experience** — Front-end and back-end experience, tools, and technologies in a clear, scannable way.
- **Showcase projects** — Highlight work via project boards (and links) in an interactive 3D space on desktop.
- **Demonstrate 3D and front-end skills** — Using React Three Fiber and modern React to show capability in 3D and interactive web development.
- **Work across devices** — 3D gallery on desktop for impact; scroll-based layout on mobile for readability and performance.

  
## Tech stack

- **React 19** + **TypeScript** + **Vite**
- **Three.js** via **@react-three/fiber** and **@react-three/drei**
- **Tailwind CSS** for styling
- **GSAP** and **@react-spring/three** for animation
